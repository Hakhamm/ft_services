FT Services

This is a System Administration and Networking project.

Introduction:
The project consists of setting up an infrastructure of different services, using Kubernetes, this is a set up to a multi-service cluster.

Services:
mysql-server with phpmyadmin control panel, ftps-server, ssh server, nginx server, wordpress ready.

cluster monitoring:
InfluxDB, Telegraf, Grafana all of the deployments are using the same address ip, which is managed by Loadbalancer (MetalLB).
All of the images used are built from scratch using alpine linux base image.
